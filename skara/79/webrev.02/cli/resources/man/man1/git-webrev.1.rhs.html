23 .TH GIT-WEBREV 1
24 .SH NAME
25 git-webrev \- render a HTML view of changes
26 .SH SYNOPSIS
27 \fIgit webrev\fR [options]
28 .SH DESCRIPTION
29 Render a HTML view of the changes between two commits.
30 .PP
31 .TP
32 \fIgit webrev\fR 
33 Creates a webrev with changes between the last two commits, and puts the resoult in a directory named 'webrev'

34 .SH OPTIONS
35 .TP
36 -r, --rev=\fIREV\fR
37 Compare against a specified revision, 
38 .PP
39 .TP
40 -o, --output=\fIDIR\fR
41 Output webrev to DIR. Defaults to 'webrev'.
42 .PP
43 .TP
44 -u, --username=\fINAME\fR
45 Use that username instead of 'guessing' one
46 .PP
47 .TP
48     --repository=\fIURL\fR
49 The upstream repository for this local repository. Defaults to the 'origin' (or 'default') URL.
50 .PP
51 .TP
52 -t, --title=\fITITLE\fR
53 The title of the webrev. Defaults to the basename of the current working directory.
54 .PP
55 .TP
56 -c, --cr=\fICR\fR
57 Include link to CR (aka bugid) in the main page
58 .PP
59 .TP
60 -b,
61 Do not ignore changes in whitespace (always true)
62 .PP
63 .TP
64 -m, --mercurial
65 Deprecated: force use of mercurial
66 .PP
67 .TP
68 -C, --no-comments
69 Don't show comments
70 .PP
71 .TP
72 -N, --no-outgoing
73 Do not compare against remote, use only 'status'
74 .PP
75 .TP
76 -v, --version
77 Print the version of this tool
78 .PP
79 .TP
80 -h, --help
81 Show help text
82 .PP
83 .SH SEE ALSO
84 \fBgit-diff\fR(1)
