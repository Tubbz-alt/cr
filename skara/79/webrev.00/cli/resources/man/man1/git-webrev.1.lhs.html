23 .TH GIT-WEBREV 1
24 .SH NAME
25 git-webrev \- render a HTML view of changes
26 .SH SYNOPSIS
27 \fIgit webrev\fR [options]
28 .SH DESCRIPTION
29 Render a HTML view of the changes between two commits.
30 .PP
31 .TP
32 \fIgit webrev\fR --base=<commit> --head=<commit>
33 Creates a webrev with changes from the base commit to the head commit
34 (i.e. base must be an ancestor of head).
35 .SH OPTIONS
36 .TP
37 --base
38 Base revision to compare against.
39 .PP
40 .TP
41 --head
42 Head revision to compare against.
43 .PP
44 .TP
45 -o <directory>, --output=<directory>
46 Output webrev to <directory>. Defaults to 'webrev'.
47 .PP
48 .TP
49 --remote=<name>
50 The remote corresponding to the upstream repository. Defaults to 'origin'.
51 .PP
52 .TP
53 --repository=<url>
54 The upstream repository for this local repository. Defaults to the 'origin' URL.
55 .PP
56 .TP
57 --title=<name>
58 The title of the webrev. Defaults to the basename of the current working directory.
