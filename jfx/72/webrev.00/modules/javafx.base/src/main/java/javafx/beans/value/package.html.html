 36 	<p>The package {@code javafx.beans.value} contains the two
 37 		fundamental interfaces {@link javafx.beans.value.ObservableValue} and {@link
 38 		javafx.beans.value.WritableValue} and all of its sub-interfaces.</p>
 39 
 40 	<h2>ObservableValue</h2>
 41 	An ObservableValue wraps a value that can be read and observed for
 42 	invalidations and changes. Listeners have to implement either {@link
 43 	javafx.beans.InvalidationListener} or {@link javafx.beans.value.ChangeListener}. To allow
 44 	working with primitive types directly a number of sub-interfaces are
 45 	defined.

 86 	<h2>WritableValue</h2>
